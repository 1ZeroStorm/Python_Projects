# Starwake Protocol

**Starwake Protocol** is a Python-based game developed primarily using the **Pygame** library. The game invites players to embark on a mission to eliminate space debris while navigating through a story-driven universe. During gameplay, players are granted unique spaceship abilities selected from loot drops, and their performance is evaluated based on the strategic choices they make.
